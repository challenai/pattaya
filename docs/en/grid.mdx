# Pattern

## Grid Pattern

Example of a grid pattern.

```js
import { drawGrid } from "@pattaya/pattaya/patterns";

const n = {
  x: 0,
  y: 0,
  postRenderCallback(ctx) {
    drawGrid(ctx, {
      width: 100,
      height: 100,
      lineWidth: 1,
      margin: 9,
      offsetX: 3,
      offsetY: 5,
      segements: undefined,
      lineColor: "#ccf",
      backgroundColor: "transparent",
    });
  }
};

graph.updateQueue(0, [[n]]);
```
