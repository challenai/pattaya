# Introduction

Welcome to **Pattaya**!

Pattaya is a modern JavaScript graphics library that provides **primitives** such as lines, squares, and bubbles,  
allowing you to create your own unique graphics effortlessly.  

Whether you need **data visualization, diagrams, architectural drawings, or custom UI components**, Pattaya has you covered.

Designed with **maintainability, simplicity, and performance** in mind, Pattaya helps you **learn less and build more**.
