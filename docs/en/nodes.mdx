# Nodes

Basic shapes include circle, rectangle, diamond and more...

## Circle

Example of a cricle.

```js
const n = {
  x: 0,
  y: 0,
  shapes: nodes.circle.shapes({ radius: 64 }, nodeStyle.normal),
},
graph.updateQueue(0, [[n]]);
```
