# 纹理

## 斜条纹

斜条纹示例

```js
import { drawStripe } from "@pattaya/pattaya/patterns";

const n = {
  x: 120,
  y: 90,
  postRenderCallback(ctx) {
    drawStripe(ctx, {
      width: 100,
      height: 100,
      lineWidth: 1,
      margin: 9,
      offset: -100,
      segements: [10, 6],
      lineColor: "#ccf",
    });
  }
};

graph.updateQueue(0, [[n]]);
```
