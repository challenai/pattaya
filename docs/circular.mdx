## Sector

Example of a sector.

```js
const n = {
  x: 180,
  y: 220,
  shapes: c.sector.shapes({ radius0: 64, radius1: 96, start: 60, end: 120 }, circularStyle.normal),
}
graph.updateQueue(0, [[n]]);
```
