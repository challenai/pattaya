## custom styles

Example of a card using custom styles.

```js
const layers = [
  [
    {
      x: 180,
      y: 120,
      shapes: card.blueprint.shape({ width: 120, height: 90, radius: 9 }, cardStyle.normal),
      contain(x, y) {
        return x > 0 && y > 0 && x < 120 && y < 90;
      },
      onMouseenter(render) {
        card.blueprint.applyStyle(this.shapes, cardStyle.active);
        render();
      },
      onMouseleave(render) {
        card.blueprint.applyStyle(this.shapes, cardStyle.normal);
        render();
      },
    },
  ],
];
graph.updateQueue(0, layers);
```
