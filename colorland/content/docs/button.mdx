## TextButton

Example of a button with text.

```js
graph.onReady(() => {
  const btn = button.basic.TextButton(240, 180, { text: "click me", radius: 8, styles, textBoundingBox: graph.boundingBox.bind(graph) });
  graph.resetGraph([[btn]]);
});
```
