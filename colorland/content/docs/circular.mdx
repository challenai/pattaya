## Sector

Example of a sector.

```js
const layers = [
  [
    {
      x: 180,
      y: 220,
      shapes: c.sector.shapes({ radius0: 64, radius1: 96, start: 60, end: 120 }, circularStyle.normal),
    },
  ],
];
graph.updateQueue(0, layers);
```

<Graph layers={sector} options={[]} height={400} styles={{ backgroundColor: background, marginBottom: "32px" }} />
