## Basic

Example of a circular pattern.

```js
const layers = [
  [
  [
    {
      x: 120,
      y: 90,
      postRenderCallback(ctx) {
        circular.drawCircular(ctx, {
          width: 100,
          height: 100,
          lineWidth: 1,
          margin: 9,
          offsetX: -20,
          offsetY: -20,
          segements: undefined,
          lineColor: "#ccf",
          backgroundColor: "transparent",
        });
      }
    },
  ],
];
graph.updateQueue(0, layers);
```
