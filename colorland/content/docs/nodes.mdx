## Circle

Example of a cricle.

```js
const layers = [
  [
    {
      x: 180,
      y: 120,
      shapes: nodes.circle.shapes({ radius: 64 }, nodeStyle.normal),
    },
  ],
];
graph.updateQueue(0, layers);
```
