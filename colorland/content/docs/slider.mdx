## Basic

Example of a slider.

```js
const layers = [
  [
    {
      x: 180,
      y: 120,
      shapes: slider.blueprint.shape({ totalLength: 200, slideLength: 120, barWidth: 3, slideWidth: 3, radius: 6, style: sliderStyle.normal }),
    },
  ],
];
graph.updateQueue(0, layers);
```
