## Basic Arrow

Example of a basic card.

```js
const layers = [
  [
    {
      x: 180,
      y: 120,
      shapes: arrow.basic.shapes({ width: 16, height: 20 }, arrowStyle.normal),
    },
  ],
];
graph.updateQueue(0, layers);
```

<Graph layers={basic} options={[]} height={400} styles={{ backgroundColor: background, marginBottom: "32px" }} />
